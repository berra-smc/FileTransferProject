
#### **TCP Client-Server File Transfer Application**



###### This project implements a TCP client-server application for sending text files.

**---------------------------------------------**

##### **FILES:**

###### server.py

###### client.py

###### lorem.txt

###### fruits.txt

###### alice.txt



**-------------------------------------------------------**

##### **HOW TO RUN**

1. ###### Open a terminal and navigate to the project directory.(cd <file path>)
2. ###### Start the server(python server.py)
3. ###### Open another terminal in the same directory.
4. ###### Run the client with a text file name(python client.py <text\_file\_name>)

###### 

###### Additional commands

###### dir/ls - listing files

###### type/cat - opening files



**--------------------------------------------------**

###### When you run python files and send the text files, server will save them and a log file in Project directory.

###### You can list or open them on command line with commands given above.
